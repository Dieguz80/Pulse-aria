## Cos'è Pulse-Aria

Pulse-Aria non è un chatbot vocale. È un **Personal AI Agent** che percepisce, ragiona e agisce sul tuo PC.

- 🎙️ **Attivazione** — doppio schiocco delle dita, wake word, o hotkey
- ☀️ **Morning Boot** — lancia le app, legge meteo, agenda ed email ogni mattina
- 🖥️ **Controllo PC** — apri app, volume, screenshot, finestre, file system
- 📧 **Email & Calendario** — Gmail, Outlook/365, IMAP, CalDAV, multi-account
- 👁️ **Visual Awareness** — "cosa vedi sullo schermo?" → screenshot + analisi AI
- 🧠 **Multi-provider AI** — Gemini, Claude, OpenAI, Ollama — porti tu la key

---

## Quickstart

1. Scarica l'installer dalla pagina [Releases](https://github.com/Dieguz80/pulse-aria/releases)
2. Installa e avvia — segui il wizard di onboarding
3. Incolla la tua API key (Gemini gratuito, Claude, o OpenAI)
4. Doppio schiocco → Aria si sveglia

**Serve aiuto con la configurazione?** Il [Setup Kit](https://pulsare.it/#kit) include video walkthrough, routine preconfigurate e guida PDF.
