## Cos'è / What is Pulse-Aria

**🇮🇹** Pulse-Aria non è un chatbot vocale. È un **Personal AI Agent** che percepisce, ragiona e agisce sul tuo PC — senza server propri. Porti tu la key del tuo provider AI preferito.

**🇬🇧** Pulse-Aria is not a voice chatbot. It's a **Personal AI Agent** that perceives, reasons and acts on your PC — no proprietary servers. You bring your own AI provider key.

## Installazione via terminale / Terminal install

**🇮🇹** Pattern in due passi: prima scarichi lo script (puoi ispezionarlo), poi lo esegui.  
**🇬🇧** Two-step pattern: first download the script (you can inspect it), then run it.

### Windows (PowerShell)

```powershell
# 1. Scarica / Download
iwr -useb https://raw.githubusercontent.com/Dieguz80/Pulse-aria/main/scripts/install.ps1 -OutFile install.ps1

# 2. (Opzionale) Ispeziona / (Optional) Inspect
notepad install.ps1

# 3. Esegui / Run
powershell -ExecutionPolicy Bypass -File install.ps1
```

### macOS (Terminal)

```bash
# 1. Scarica / Download
curl -fsSL -o install.sh https://raw.githubusercontent.com/Dieguz80/Pulse-aria/main/scripts/install.sh

# 2. (Opzionale) Ispeziona / (Optional) Inspect
cat install.sh

# 3. Esegui / Run
bash install.sh
```

**🇮🇹** Lo script rileva automaticamente architettura (ARM/Intel su macOS), scarica l'ultima release e avvia l'installer.  
**🇬🇧** The script automatically detects architecture (ARM/Intel on macOS), downloads the latest release and launches the installer.

## Primo avvio / First launch

### Windows SmartScreen

**🇮🇹** Al primo avvio Windows potrebbe mostrare "Windows ha protetto il PC" perché l'app non è ancora firmata con un certificato EV (in arrivo con v0.6).

1. Clicca **"Ulteriori informazioni"**
2. Clicca **"Esegui comunque"**

**🇬🇧** On first launch Windows may show "Windows protected your PC" because the app isn't yet signed with an EV certificate (coming with v0.6).

1. Click **"More info"**
2. Click **"Run anyway"**

### macOS Gatekeeper

**🇮🇹** Al primo avvio macOS potrebbe rifiutare l'apertura ("Pulse-Aria non può essere aperta perché lo sviluppatore non può essere verificato").

1. Apri **Impostazioni di Sistema → Privacy e sicurezza**
2. Scorri in fondo, trova "Pulse-Aria è stata bloccata" e clicca **"Apri comunque"**
3. Conferma con la password macOS

**🇬🇧** On first launch macOS may refuse to open ("Pulse-Aria cannot be opened because the developer cannot be verified").

1. Open **System Settings → Privacy & Security**
2. Scroll down, find "Pulse-Aria was blocked" and click **"Open Anyway"**
3. Confirm with your macOS password

## Privacy

**🇮🇹** Pulse-Aria funziona principalmente sul tuo dispositivo. Wake word, controllo app, monitor sonno e configurazione restano locali. I dati escono solo verso il provider AI che scegli (Gemini / Claude / OpenAI), solo per rispondere alle tue richieste, e solo quando i toggle privacy lo permettono.

**🇬🇧** Pulse-Aria runs mainly on your device. Wake word, app control, sleep monitor and configuration stay local. Data leaves only toward the AI provider you choose (Gemini / Claude / OpenAI), only to answer your requests, and only when privacy toggles allow it.

4 toggle granulari / 4 granular toggles:

- **Vision** (cattura schermo / screen capture) — off by default
- **Allegati file / File attachments** — off by default
- **Contesto finestra attiva / Active window context** — on by default
- **Google Search grounding** — on by default
